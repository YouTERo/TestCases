# Инструкция по использования автоматизированных тестов

## Требования
- Python 3.14
- pip

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости
   ```bash
   pip install -r REQUIREMENTS.txt
3. Запустите тесты из папки проекта
   ```bash
   pytest TEST_API.py -v
