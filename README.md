# Инструкция по использования автоматизированных тестов

## Требования
- Python 3.14
- pip

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости
   ```bash
   pip install -r requirements.txt
3. Запустите тесты из папки проекта
   ```bash
   pytest test_api.py -v
